# Programmeren Gevorderd 2 Code Repo

- Copy-paste van de code uit de les.
- Een folder per lesdag.

Tijdens de lessen wordt er aan live coding gedaan. Dit heeft als gevolg dat de code uit de les niet exact overeen zal komen met de cursus.

Laat je hierdoor niet in de war brengen. Je zal snel ondervinden dat deze afwijkingen zeer oppervlakkig zijn en het geheel conceptueel identiek is aan de cursus. Immers: de lessen volgen de cursus, niet omgekeerd.
